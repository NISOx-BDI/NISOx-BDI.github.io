Introduction

APACE is a Matlab script which performs Accelerated Permutation inference for ACE models. It is partitioned into 4 main parts: the preparation for the analysis, permutation analysis for computing p-values, bootstrapping analysis for generating confidence intervals (CIâ€™s) and aggregate heritability analysis. It offers 4 optional inference approaches: voxelwise inference, cluster-wise inference, summary measure inference, and the aggregate inference (The default is to implement all inferences, but the users can easily select only the desired inferences).

APACE can be downloaded from here or alternatively the APACE GitHub can be accessed here. Also see the manual and example code (both of these are also in the ZIP file on the download page).
