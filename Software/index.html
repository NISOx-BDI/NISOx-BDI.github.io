Professor Thomas Nichols and the NISOx group have contributed towards the development of two neuroimaging software analysis programs, FSL & SPM.
  • FSL stands for FMRIB's Software Library, and is a product of the analysis group at Oxford University's Centre for Functional Magnetic Resonance Imaging of the Brain. FSL is a collection of C++ programs sewed together with bash scripts and tcl/tk GUIs.
  • SPM stands for Statistical Parametric Mapping, and is the product of the methods group at University College London's Wellcome Trust Centre for Neuroimaging ("The FIL"). SPM is a collection of Matlab scripts and Matlab-linked C code.

In addition Prof. Thomas Nichols also has created and contributed towards several distinct add-on toolboxes. These include the following:

  • Randomise
    The component of FSL that provides nonparametric permutation-based inference.
  • SnPM
    A toolbox for SPM provides nonparametric permutation-based inference. (Releases documented: SnPM13, SnPM8)
  • SPMd
    A toolbox or SPM that provides assumption-checking and data exploration tools.
  • FDR
    A more powerful approach to multiple testing (was an extension, now integrated into SPM).
  • SwE
    An SPM extension for longitudinal and repeated measures data.
  • NINGA
    A toolbox implementing fast and powerful genome wide association analysis of dense genetic data with high dimensional imaging phenotypes.
  • FPHI
    A toolbox for Heritability Inference for Related Subjects Based on Neuroimaging Phenotypes.
  • BSGLMM
    A toolbox implementing the Bayesian Spatial Generalised Linear Mixed Model.
  • BHPGM
    A toolbox implementing the Bayesian Hierarchical Poisson Gamma Model.
  • BHICP
    A toolbox implementation of Bayesian Hierarchical Independent Cox Processes.
  • APACE
    A tool for fast heritability inference.

In addition, for extra code excerpts, snippets and tips, please refer to the NISOx blog.
