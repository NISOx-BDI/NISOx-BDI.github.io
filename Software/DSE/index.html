DSE Decomposition

DSE is a toolbox for variance decomposition of 4D data sets in order to diagnose spatio-temporal corruptions. Please direct any questions about this work to Soroosh Afyouni & Tom Nichols.

  • Manuscript:
    Afyouni, Soroosh, and Thomas E. Nichols. "Insight and Inference for DVARS." NeuroImage (2018).
    DOI: 10.1016/j.neuroimage.2017.12.098
  • Github:
    Source code and full release details can be found here.
